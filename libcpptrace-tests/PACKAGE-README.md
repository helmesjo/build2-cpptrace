# libcpptrace-tests - Portable C++ stacktrace library (tests)

This is a `build2` package containing the test suite for the
[`cpptrace`](https://github.com/jeremy-rifkin/cpptrace) library. Currently
only the upstream unit tests (using `gtest`/`gmock`) are included.


## Importable targets

This package exports no targets.


## Configuration variables

This package provides no configuration variables.
